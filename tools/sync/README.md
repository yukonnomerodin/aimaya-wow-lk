# aimaya-wow-lk

Это «управляющий» репозиторий: он не дублирует код, а помогает одним командой держать на ПК нужные репы в `C:\_azerothcore`.

## Синхронизация

Из папки `C:\_azerothcore\aimaya-wow-lk`:

```powershell
.\tools\sync\sync.ps1
```

Что делает:
- если репы нет — скачает (`git clone`)
- если репа уже есть — обновит (`git pull --ff-only`)

## Куда кладёт

По умолчанию — в `C:\_azerothcore` (на уровень выше этого репозитория).

## Список реп

Файл `tools\sync\repos.json`.
